    Settings

    Settings are saved to local vmix-bridge/config.json when you click Save. The browser also keeps a localStorage recovery backup and will restore it into the form if the server config is missing or blank.

    vMix research note: API.Function(functionName, input, value, ...) maps to the HTTP Web API. The bridge sends the mapping value raw to the chosen function. For your production setup, that value can be the camera number (for example 100). If a different input expects Index,Input syntax as described in the shortcut reference, enter that raw string instead. Set Unmapped fallback value to a safe camera/blank value so unmapped players do not leave vMix stuck on the previous mapped player.

    Nickname SetText

    Optional second vMix call for the displayed nickname. The nickname value is the current spectated name and is URL-encoded safely for spaces, quotes, symbols, and Russian characters.

    Username → Value mapping table

    Username cells are searchable. Rows turn sapphire/amber when the selected username's team is known. The Nickname input column sends SetText to that vMix input with SelectedName=Nickname.Text after a row edit settles. Use Send all row nicknames to resend every row.
